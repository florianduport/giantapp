AppWebcore
======
Ce dossier devra contenir le coeur de l'application générique. 

Application
======
Cette application devra être compilée N fois avec phonegap pour chaque client.

Webcore
======
Ce dossier contiendra le site qui sera contacté par les N applications. Ce site retournera le contenu des pages. Le contenu de chaque application pourra donc être modifié sans avoir besoin de télécharger la mise à jour de l'application chez les clients.
